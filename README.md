# Windbnb - Citlalli
Este proyecto muestra alojamientos en Finlandia y permite filtrarlos por ciudad y número de huéspedes.

##  Demo

Puedes ver el proyecto en línea aquí:

https://windbnb-citlalli.vercel.app

##  Tecnologías usadas

- HTML
- CSS
- JavaScript
- Vite
- TailwindCSS
- Vercel (deploy)

##  Estructura del proyecto

src/
 main.js
 stays.js
 utilis.js
 style.css
 images/
 icons/

##  Funcionalidades

- Mostrar alojamientos disponibles
- Filtrar por ciudad
- Filtrar por número de huéspedes
- Mostrar rating con estrellas
- Diseño responsive
